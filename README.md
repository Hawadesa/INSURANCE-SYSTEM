Insurance Policy Management System (Python & R)
Project Overview
This is a simple Python and R based system to manage:
- Policyholders
- Products
- Payments
 Features
- Register, suspend, and reactivate policyholders
- Create, update, and suspend products
- Process payments, send reminders, and apply penalties
 Technologies Used
•	Python 
•	R 
•	Missing exported files
 How to Run
1. Ensure all files are in the same directory
2. Run:
   python main.py
